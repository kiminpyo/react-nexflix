```
netflix페이지를 클론한 홈페이지입니다. the moiveDB API키를 이용해 영화 정보를 불러옵니다. 
메인페이지에서 영화를 검색할 수 있고, 상세보기를 조회할 수 있습니다.
```

### 💻 실행 방법

1.  라이브러리를 설치합니다.

```
npm install
```

2.  프로젝트를 실행합니다.

```
 npm start
```
